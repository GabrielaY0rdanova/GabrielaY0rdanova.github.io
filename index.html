Projects

PharmaMarket ETL

A full ETL pipeline that extracts data from CSV files, parses complex multi-value fields, and loads it into a structured SQL Server database. Covers drug classes, generics, manufacturers, dosage forms, indications, and medicines — including package sizing, container pricing, and a many-to-many relationship between generics and indications.
